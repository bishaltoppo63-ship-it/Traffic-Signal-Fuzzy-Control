# Traffic-Signal-Fuzzy-Control
# 🚦 Traffic Signal Control using Fuzzy Logic

## 📌 Project Overview
This project implements an intelligent traffic signal control system using Fuzzy Logic. 
The system dynamically adjusts signal timings based on traffic density.
